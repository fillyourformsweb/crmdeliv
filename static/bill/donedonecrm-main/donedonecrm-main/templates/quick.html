Quick Help Guide

📋 Order Processing Steps:

  1. Select services - Click on service cards
  2. Enter quantity - Use the popup to select quantity
  3. Enter phone number - Customer name will auto-fill if in database
  4. Choose payment method
  5. Enter amount received
  6. Click "Complete Order"

🔢 Quantity Controls:

  • Click service card to select quantity
  • Use +/- buttons to adjust quantity
  • Change quantity later in selected items list
  • Remove items with trash icon

👤 Customer Lookup:

When you enter a phone number, the system automatically checks if the customer exists in the database and fills their name automatically.
