Bulk Import Orders

Import multiple orders from an Excel file at once

How to Use Bulk Import
  1. Prepare an Excel file with your order data
  2. Ensure required columns are present (customer_name, receiver_name, receiver_pincode, etc.)
  3. Upload the file below
  4. The system will automatically create orders and generate receipt numbers
Required Excel Columns
Required Fields
  • customer_name - Sender/Customer name
  • customer_phone - Sender phone number
  • receiver_name - Recipient name
  • receiver_pincode - Recipient pincode (6 digits)
  • receiver_state - Delivery state
  • receiver_address - Recipient address
  • weight - Package weight in kg
Optional Fields
  • order_type - walkin or client (default: walkin)
  • customer_email - Sender email
  • customer_address - Sender address
  • receiver_phone - Recipient phone
  • receiver_city - Recipient city
  • package_description - What's being shipped
  • number_of_boxes - Number of boxes (default: 1)
  • receipt_mode - standard/prime/express (default: standard)
  • special_instructions - Special handling notes
Upload Excel File
Maximum file size: 10MB. Supported formats: Excel (.xlsx), CSV
Sample Excel Format
customer_name customer_phone receiver_name receiver_phone receiver_address receiver_city receiver_state receiver_pincode weight package_description
John Doe 9876543210 Jane Smith 9876543211 123 Main St Mumbai Maharashtra 400001 0.5 Documents
ABC Company 9876543212 Receiver Name 9876543213 456 Business Ave Delhi Delhi 110001 2.0 Electronics
Tips for Successful Import
  • Check Pincode Format: All pincodes must be exactly 6 digits
  • Required Data: Missing customer name, receiver name, or pincode will skip that row
  • Weight Format: Use decimal format (0.5, 1.0, 2.5, etc.)
  • States: Use the complete state name (e.g., "Maharashtra", not "MH")
  • Column Names: Column names are case-sensitive - use exactly as shown
  • Receipt Numbers: System automatically generates unique receipt numbers - don't include them in the file
  • Phone Numbers: Enter phone numbers without spaces or special characters
